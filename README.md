## Commit Rule

### 1️⃣Study Format

```
git commit -m "[Keyword]: [(#Issue)] 제목"
```

- 💡예시: `[docs]: [(#1)] 스택`

#### Keyword

<table>
<tHead>
<tr>
<th>상태</th>

```
git commit -m "[Keyword]: [(#Issue)][Platform][Level] 제목"
```
